# RubyMine Local Development
Crib Sheet
## Setup
``` shell
./.rubymine/setup.sh
```
## Docker Compose Build and Up
```shell
docker-compose build --no-cache
docker-compose up -d
```
## RubyMine Settings
Languages & Frameworks > Ruby SDK and Gems
 + Remote Interpreter or Version Manager
## Docker Compose Down
```shell
docker-compose down
```
## Teardown
``` shell
./.rubymine/teardown.sh
```
